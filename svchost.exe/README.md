### svchost.exe
> Short for Service Host.\
**NOTE: You may or may not need a kernel driver to read and or write memory from this process. It will depend on what service you're trying to read from.

### Explanation
> Service Host is crucial for Windows, acting like a container to run a variety of Windows services including but not limited to files, audio, networking, logs, and more. In return, the system saves its resources as it doesn't have to run each process individually.
